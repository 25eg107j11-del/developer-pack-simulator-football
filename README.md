# Football Pack Simulator

A fun little web app I built to simulate opening football (soccer) card packs. It's got the whole pack animation sequence: shaking, bursting, and even walkouts for the rare ones! Built this mainly to play around with React and some neat CSS animations.

## Features
- **Pack Animations**: Smooth shake, burst, and reveal phases.
- **Walkouts**: If you pack a rare card (or better), it triggers a special walkout sequence showing the country, position, rating, and name before the big reveal.
- **Card Rarities**: Bronze, Silver, Gold, Rare, Ultra-Rare, and Icons. Each has its own styling and flare gradient.
- **Responsive Design**: Works well on desktop and mobile.

## Tech Stack
- Next.js (App Router)
- React
- Vanilla CSS for styling and animations

## Running Locally

To get this running on your own machine:

```bash
npm install
npm run dev
```

Then pop open `http://localhost:3000` in your browser.
